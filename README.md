# 🚁 Drone Swarm Defense Simulator (OS-Based System)

## 📌 Overview
This project is a real-time simulation of an autonomous drone swarm defense system.  
It demonstrates how multiple drones coordinate to intercept and neutralize an enemy swarm.

The project also integrates **Operating System concepts** such as process scheduling, process states, and inter-process communication.

---

## 🎯 Objectives
- Simulate coordinated swarm engagement
- Implement intelligent targeting and prediction
- Apply swarm intelligence (boids behavior)
- Integrate OS concepts like scheduling and resource management
- Visualize system performance in real-time

---

## ⚙️ Features

### 🤖 AI & Swarm Intelligence
- Predictive targeting (future position estimation)
- Time-To-Impact (TTI) based scoring
- Cohesion, Alignment, Separation (Boids model)

### ⚡ Performance Optimization
- Object pooling (memory optimization)
- FPS-based adaptive rendering
- Particle limit control

### 🖥️ Operating System Concepts
- Each drone acts as a **process**
- **Round Robin Scheduling** with time quantum
- Process states:
  - READY
  - RUNNING
  - WAITING
- Process Control Block (PCB)
- Context switching
- Inter-Process Communication (IPC)

### 📊 Visualization & UI
- Military-style HUD interface
- Real-time analytics panel
- OS Scheduler panel
- Event logs
- Minimap view
- Advanced AI visualization mode

---

## 🧠 System Workflow

1. Enemy drones spawn in waves  
2. Friendly drones predict enemy movement  
3. Threat scoring is calculated  
4. Scheduler assigns CPU time (Round Robin)  
5. Swarm coordination executes interception  
6. System updates stats and logs in real-time
